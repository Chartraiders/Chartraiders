## What Is Fantasy Trading?

Fantasy Trading is Chart Raiders' coined term for a competitive format in which players use skill to maximize a fictional bankroll by analyzing real-world market data — competing against other players for rank within a time-limited match. It is the fantasy sports model applied to financial market analysis.

**Players never buy, sell, or hold any securities — registered or unregistered.** Players analyze publicly available market data and make decisions within a fictional environment. Payouts are distributed by player rank among peers — not tied to the direction or value of any underlying asset.

\---

## Game of Skill

Chart Raiders is a Game of Skill. Outcomes are determined by trading knowledge, pattern recognition, quantitative reasoning, and strategic decision-making.

**What makes it a Game of Skill:**

* Players with stronger analytical skills consistently outperform players with weaker skills over time
* The Chart Raiders Skill Tree is built to develop and reward the competencies that determine match outcomes
* All players begin each match with the same fictional bankroll ($100) — standardized starting conditions ensure outcomes reflect skill, not real-world wealth
* Players make multiple trading decisions over the course of each match — the aggregate of many skill-based decisions determines final rank

\---

## Contest Structure

Chart Raiders operates as a **peer-to-peer contest platform**:

* Players compete against each other for rank
* Chart Raiders charges a platform fee and does not participate as a player, set odds, take positions, or act as a counterparty
* All players are fully informed **before the match starts** of: total entrants, complete prize pool and distribution breakdown, and the ticker being analyzed
* Prize distribution is based on player rank — the intervening variable is player skill and decisions, not security performance
